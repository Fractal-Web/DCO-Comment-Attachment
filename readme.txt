Allows your visitors to attach files with their comments

== Description ==
DCO Comment Attachment allows your visitors to attach files with their comments, such as documents and images.

== Installation ==
1. Upload `dco-comment-attachment` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
